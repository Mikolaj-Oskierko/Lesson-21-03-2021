Styling links and lists

Anchor links

  • Links can take on all of the usual styles as well as text-decoration
  • Links by default are inline, so if you wants to chage the width, you have to use display:block; or display:inline-block;.

Buttons

  • Many designers try to make their links look like buttons.
  • We want to avoid that, we want to be semantic.
  • If you want a button use button HTML tag.

States

  • Some links are blue, some are purple, etc. Why?
    • a:link - normal link
    • a:visited - visited
    • a:hover - hover activity, not on touch screens.
    • a:focus - TAB jumping
    • a:active - clicked

Precedence of rules

  • a:hover MUST come after a:link and a:visited
  • a:active MUST come after a:hover
